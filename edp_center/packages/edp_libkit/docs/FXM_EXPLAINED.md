# FXM Files Explained

## Overview
FXM (Foundry Exchange Model) files contain foundry-specific technology information.

## Content
- Process technology parameters
- Metal stack definitions
- Via definitions
- Design rules

## File Format
Typically XML or proprietary binary format containing:
- Layer definitions
- Spacing rules
- Width/length constraints
- Via enclosure rules

## Usage
- Technology file generation
- P&R tool setup
- Physical verification setup
- Custom tool integration

## Foundry Specific
Each foundry provides their own FXM format and content structure.
