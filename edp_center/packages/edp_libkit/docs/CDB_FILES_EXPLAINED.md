# CDB Files Explained

## Overview
Calibre Database (CDB) files contain physical verification rules and models.

## File Types
- `.cdb` - Calibre rule database
- `.svrf` - Standard Verification Rule Format (source)

## Content
- DRC (Design Rule Check) rules
- LVS (Layout vs Schematic) rules
- Antenna rules
- Density rules

## Usage
Used in Calibre for:
- Physical verification
- DRC checking
- LVS verification
- Parasitic extraction setup

## Integration
Typically referenced in verification flows and P&R tools for rule checking.
